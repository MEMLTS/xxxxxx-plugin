## 📦 安装方法

<summary>👉 详细步骤</summary>

1. 在Yunzai根目录下执行
```bash
git clone --depth=1 https://github.com/MEMLTS/xxxxxx-plugin.git ./plugins/xxxxxx-plugin/
```

1. 安装插件依赖
```bash
pnpm install --filter=xxxxxx-plugin
```

1. 重启Yunzai-Bot

## ⚙️ 配置说明
